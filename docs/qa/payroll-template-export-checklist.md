# 劳务工资代发模板系统 QA 清单

- [ ] 普通录入人员只能看到自己创建的人员、单位、批次、导出记录和日志。
- [ ] 管理员可以看到全部人员、单位、批次、导出记录和日志。
- [ ] 新建工资批次时，可以批量选择人员库人员。
- [ ] 人员库没有的人员，可以在工资明细表中逐行录入。
- [ ] 保存批次后，手工录入的新人员会自动进入人员库。
- [ ] 同一身份证号再次保存时，不新增重复人员，只更新可变信息。
- [ ] 工资明细金额变化后，总人数和总发金额实时刷新。
- [ ] 页面能显示缺银行信息人数和缺联行号人数。
- [ ] 同一工资批次可以选择不同付款单位和不同银行模板导出。
- [ ] 建设银行导出文件字段顺序匹配 `建设银行工资模版.xlsx`。
- [ ] 中国银行导出文件字段顺序匹配 `中国银行工资模版.xls`。
- [ ] 每次导出都新增导出记录，不覆盖历史导出记录。
- [ ] 批次导出后仍可继续修改并再次导出。
- [ ] 新增、修改、删除、导出动作都写入操作日志。
- [ ] 页面整体清爽、表格不拥挤、总金额醒目、录入过程顺手。
